overview
Bun Bun Bakery was a project completed for Programming Usable Interfaces at Carnegie Mellon University. The goal of the project was to design and develop an online shopping website for a fictional business selling cinnamon buns. Functionality was implemented via vanilla HTML, CSS, and Javascript.

prototyping and user testing
Initially, I created three low-fidelity paper prototypes for initial user testing. The prototypes had variations in navigation, information display, and shopping interactions. These paper prototypes were tested with a peer, resulting in the following insights:

01. Multiple navigation options presented in a conventional manner are important -- i.e. clear navigation bars and back buttons.
02. Users like to view their options -- being able to see all available glazes and flavors was preferred to drop-down menus.
03. Separation between user interactions and descriptive information is important to help the user locate information.
With these preferences in mind, I moved on to create high-fidelity screens and a site map for the Bun Bun Bakery website. Design priorities were ease of navigation, user comfort and flexibility, and creating an aesthetic experience that reflected Bun Bunâ€™s personality as a small business.
implementation
A few selected pages of the website were implemented as an exercise to learn and practice HTML, CSS, and Javascript. No libraries were used. The website was not responsive. Key functions included browsing the menu, allowing the user to customize their blackberry bun via the product detail page, and automatically updating the cart page.
